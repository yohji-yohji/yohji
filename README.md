# Yoji Motion Resume

这一版不是普通静态简历页，而是参考 `cappen.com` 的实现思路重构的个人品牌站。

核心技术栈：

- `Vite`
- `GSAP`
- `Lenis`
- `Three.js`

核心表达方向：

- 视觉设计
- AI Coding
- 产品能力

## 本地开发

```bash
npm install
npm run dev
```

默认地址：

```text
http://127.0.0.1:7878
```

## 构建

```bash
npm run build
```

## 预览构建结果

```bash
npm run preview
```
